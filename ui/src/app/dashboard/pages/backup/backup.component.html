Backup

Instructivo para backup y recuperacion de datos

Backup
  1. Abrir el cmd como administrador e ir al path bin donde esta instalado postgres
    cd C:\Program Files\PostgreSQL\9.6\bin
  2. Formato del comando:

    pg_dump -U 'usuario' -W -h 'host' 'database' > 'backup.sql'

    Completar los literales que se encuentran entre comillas simples y ejecutar el comando:
    3. pg_dump -U postgres -W -h localhost agronome > agronomeBackup.sql
  3. Se crea el archivo agronomeBackup.sql en la carpeta C:\Program Files\PostgreSQL\9.6\bin , en donde se visualiza el script sql
Backup de una tabla
  1. Abrir el cmd como administrador e ir al path bin donde esta instalado postgres
    cd C:\Program Files\PostgreSQL\9.6\bin
  2. Formato del comando:

    pg_dump -U 'usuario' -W -h 'host' 'database' -t 'tablename' > 'backup.sql'

    Completar los literales que se encuentran entre comillas simples y ejecutar el comando:
    3. pg_dump -U postgres -W -h localhost -t actividad agronome > actividadbackup.sql
Recuperación
  1. Formato del comando:

    psql -U 'usuario' -W -h 'host' 'database' < 'backup.sql'

    Lanzar:

    psql -U 'usuario' -W -h 'host' 'database' < 'backup.sql'

    psql -U postgres -W -h localhost agronome < agronomeBackup.sql
